\streszczeniepracy{
Celem pracy jest zaprojektowanie i~zaimplementowanie autorskiego Bliźniaka Cyfrowego (Digital Twin) dwukierunkowego przekształtnika prądu stałego (typu Buck-Boost) w~języku Python. Projekt zakłada przeniesienie zamkniętych modeli inżynierskich do otwartej architektury programistycznej, z~wykorzystaniem numerycznych metod rozwiązywania równań różniczkowych. Zbudowane środowisko zostanie zintegrowane z~algorytmami sztucznej inteligencji, w~szczególności z~metodą Optymalizacji Rojem Cząstek (PSO), w~celu zautomatyzowanego strojenia bezmodelowego algorytmu sterowania w~stanach skończonych (Bang-Bang). Praca ma charakter stricte programistyczny i~optymalizacyjny, a~jej zwieńczeniem będzie empiryczna walidacja wyników z~komercyjnym oprogramowaniem PSIM.

\slowakluczowe{Bliźniak Cyfrowy, Python, Optymalizacja Rojem Cząstek (PSO), Przekształtnik Buck-Boost, Inżynieria Oprogramowania, Sterowanie bezmodelowe}

\thesisabstract{
The aim of this thesis is to design and implement a~proprietary Digital Twin of a~bidirectional DC-DC converter (Buck-Boost type) in the Python programming language. The project involves transferring closed engineering models into an open software architecture using numerical methods for solving differential equations. The developed environment will be integrated with artificial intelligence algorithms, specifically Particle Swarm Optimization (PSO), for the automated tuning of a~model-free finite-state control algorithm (Bang-Bang). The thesis is strictly focused on software engineering and optimization, culminating in an empirical validation of the results against commercial PSIM software.
